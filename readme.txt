Инструкция по запуску:
1. docker build -t express-money-front .
2. docker images
3. docker run -d --rm -p 8000:8000 --name express-money-front express-money-front
4. docker ps

Некоторые страницы находятся в разработке, чтобы их глянуть:
1. добавьте /tech-page к baseurl
2. нажмите на кнопку с названием страницы
